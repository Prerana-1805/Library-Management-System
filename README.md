# Library Management System

This project is developed using Java and OOP concepts.

Features:
- Add Book
- Show Books
- Search Book
- Issue Book
- Return Book
- Duplicate Book ID Validation

Concepts Used:
- OOP Concepts
- ArrayList
- HashMap
- Exception Handling
- Java 8 Streams
- Lambda Expressions

Files:
- Book.java
- User.java
- Library.java
- InvalidBookOperationException.java
- Main.java

How to Run:
1. Open project in Eclipse
2. Run Main.java
3. Use menu options in console
